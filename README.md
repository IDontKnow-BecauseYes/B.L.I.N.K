# B.L.I.N.K
B.L.I.N.K – Behavior Log Insight Notification Keep-up Um app para acompanhar sua rotina. Ele registra Comportamento (Behavior), organiza Registros (Log), gera Percepções (Insight), envia Notificações (Notification) e ajuda na Manutenção do ritmo (Keep-up). Simples, direto e focado em manter o usuário firme nas metas.
